A-Frame

A-Frame adalah kerangka kerja web sumber terbuka yang di buat untuk membangun pengalaman realitas virtual (VR). A-Frame dikelola oleh pengembang dari Supermedium (Diego Marcos, Kevin Ngo) dan Google (Don McCurdy). Kerangka kerja A-Frame mengunakan sistem komponen entitas untuk Three.js di mana pengembang dapat membuat adegan 3D dan WebVR menggunakan HTML. Saat ini A-Frame masih terus berkembang untuk menjadi salah satu VR terbesar dalam dunia pengembangan VR.

A-Frame mendukung sebagian besar headset VR seperti Vive, Rift, Windows Mixed Reality, Daydream, GearVR, Cardboard, Oculus Go, dan bahkan dapat digunakan untuk augmented kenyataan. Referensi lengkap terkait A-Frame anda dapat mengunjungi situs resmi A-Frame link.

AR.Js

AR.js adalah pustaka Javascript pengembangan dari ARToolKit, salah satu SDK AR pertama untuk seluler. AR.Js membuat perubahan pada dunia reality berbasis Web, Pada dasarnya AR.Js bertujuan untuk memberikan pustaka Javascript AR dengan program simulasi yang ringan untuk di jalankan.

Referensi lengkap terkait AR.Js anda dapat mengunjungi situs Github org AR.Js link.

A-Frame basic view

Contoh project yang dibuat dengan A-Frame dan AR.Js basic, dengan mejalankan HTML dan memberikan akses kamera, ketika preset yang di tentukan terdeteksi maka program akan menampilkan gambar bumi text standar.

Html

              
                <body style="margin: 0px; overflow: hidden">

                  <a-scene embedded arjs="sourceType: webcam;">
                    <a-entity
                      rotation="-90 0 0"
                      geometry="primitive: plane; width: 4; height: auto"
                      material="color: blue"
                      text="value: Bumi \n Menampilkan gambar bumi dasar"
                    >
                    </a-entity>
                    <a-image src="images/earth-flat.jpg" rotation="-90 0 0"></a-image>
                    <a-marker-camera preset="hiro"></a-marker-camera>
                  </a-scene>
              
                </body>

Html

Html

                          position="0 0 0"
                          scale="0.2 0.2 0.2"
